# Starter notification templates

These HTML snippets are intentionally simple. They are safe fallbacks for local development before Creator Studio templates are configured.

## `saas-welcome`

```html
<h1>Welcome to Your Product</h1>
<p>Your workspace is ready. Sign in any time to continue building.</p>
```

## `saas-admin-invite`

```html
<h1>You were invited to help manage Your Product</h1>
<p>An administrator invited you to review the admin dashboard and help manage users.</p>
```

## `saas-password-changed`

```html
<h1>Your password was changed</h1>
<p>If this was not you, contact support immediately and rotate your credentials.</p>
```

## `saas-account-deleted`

```html
<h1>Your account has been deleted</h1>
<p>We're sorry to see you go. This message confirms that your account data was removed.</p>
```
